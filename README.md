# 🤖 Voice AI Agent

An intelligent **Voice AI Agent** that understands user intent and executes tools such as **search and research** to generate useful outputs.

The system allows users to interact using **text or voice input** and processes queries using an **agent-based architecture**.

---

## 📌 Project Overview

This project demonstrates how **AI agents** can automate tasks like:

🔎 Information retrieval  
📚 Research summarization  
🧠 Intelligent query processing  
💾 Automatic output storage  

The agent analyzes user input, detects the intent, and selects the appropriate tool to perform the task.

---

## ✨ Features

✔️ Intent detection for user queries  
✔️ Web search capability  
✔️ Research summarization  
✔️ Voice + text input support  
✔️ Automatic output file saving  
✔️ Modular AI agent architecture
